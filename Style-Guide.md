Braces: own line
```
if (true)
{
    return;
}
```
Semicolons: yes
```
console.log("look at this line");
```
Indentation: 4 spaces
```
if (true)
{
    console.log("indented line");
}
```
If braces: always
```
if (true)
{
    console.log('one line block, but has braces');
}
```
Variable declaration: One per line
```
var a = 1;
var b = 2;
```
Trailing commas in object literals: yes
```
var obj = 
{
    first: 1,
    last: 2,
}
```
